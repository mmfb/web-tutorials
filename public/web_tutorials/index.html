Objective

These tutorials address the topic of building a restfull web server using Node.js/Express.js

We will be creating a Node.js/Express.js project with a REST api connected to a database and web page server with HTML, CSS and Javascript

We will use static web pages and dynamic client side pages that use AJAX calls (using JQuery) to make requests to the REST api
